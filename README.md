# 已关闭网页查看器 (Closed Tabs Viewer)

一个简洁实用的 Firefox 浏览器扩展，用于查看和管理被关闭的网页历史记录。

## 功能特点

- 📝 **自动记录** - 自动记录所有关闭的标签页
- ⏰ **时间显示** - 显示每个网页的关闭时间（具体时间 + 相对时间）
- 📅 **日期分组** - 按日期分组显示（今天、昨天、具体日期）
- 🔍 **快速恢复** - 点击即可重新打开已关闭的网页
- 🗑️ **灵活管理** - 支持删除单条记录或清空全部
- 💾 **本地存储** - 最多保存 100 条记录

## 安装方法

### 方法一：临时加载（开发模式）

1. 下载或克隆本项目到本地
2. 打开 Firefox 浏览器
3. 地址栏输入 `about:debugging`
4. 点击左侧 **此 Firefox**
5. 点击 **临时载入附加组件**
6. 选择项目中的 `manifest.json` 文件

### 方法二：打包安装

1. 将项目文件打包为 `.xpi` 文件
2. 在 Firefox 中打开 `about:addons`
3. 点击齿轮图标 → **从文件安装附加组件**
4. 选择打包好的 `.xpi` 文件

## 使用说明

1. 点击浏览器工具栏中的扩展图标
2. 查看已关闭的网页列表
3. 点击任意条目或"打开"按钮恢复网页
4. 点击"删除"按钮移除单条记录
5. 点击"清空全部"清除所有历史记录
