# GitHub 上传步骤指南

## 前置准备

### 1. 安装 Git
- 下载地址: https://git-scm.com/download/win
- 安装时使用默认设置即可
- 安装完成后重启电脑或重新打开终端

### 2. 配置 Git 身份信息
打开 PowerShell，运行以下命令（替换为你的信息）：

```powershell
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub注册邮箱"
```

### 3. 在 GitHub 创建仓库
1. 登录 https://github.com
2. 点击右上角 **+** → **New repository**
3. Repository name: `closed-tabs-viewer`
4. 选择 Public 或 Private
5. **不要勾选** "Add a README file"
6. 点击 **Create repository**

---

## 上传命令

在项目目录下打开 PowerShell，依次执行以下命令：

### 第一步：初始化仓库
```powershell
git init
```

### 第二步：添加所有文件
```powershell
git add .
```

### 第三步：创建第一次提交
```powershell
git commit -m "初始提交：Firefox已关闭网页查看器扩展"
```

### 第四步：关联远程仓库
将下面的 `你的用户名` 替换为你的 GitHub 用户名：

```powershell
git remote add origin https://github.com/你的用户名/closed-tabs-viewer.git
```

### 第五步：推送到 GitHub
```powershell
git branch -M main
git push -u origin main
```

---

## 如果推送失败

### 方法1：使用 Personal Access Token（推荐）

GitHub 现在需要使用 Token 进行认证：

1. 登录 GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. 点击 **Generate new token (classic)**
3. 勾选 `repo` 权限
4. 点击 **Generate token**
5. 复制生成的 token
6. 推送时使用 token 作为密码

推送命令：
```powershell
git push -u origin main
```
- Username: 你的GitHub用户名
- Password: 粘贴刚才的 token

### 方法2：使用 SSH

1. 生成 SSH 密钥：
```powershell
ssh-keygen -t ed25519 -C "你的邮箱"
```

2. 查看公钥：
```powershell
cat ~/.ssh/id_ed25519.pub
```

3. 将公钥添加到 GitHub：Settings → SSH and GPG keys → New SSH key

4. 使用 SSH 地址：
```powershell
git remote set-url origin git@github.com:你的用户名/closed-tabs-viewer.git
git push -u origin main
```
